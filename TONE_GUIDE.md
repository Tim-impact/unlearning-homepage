# 언러닝컴퍼니 톤앤보이스 가이드

> 이후 홈페이지·소개서·블로그·메일·제안서를 쓸 때 일관된 결을 유지하기 위한 내부 기준 문서입니다.
> 기준 모델은 **루트임팩트**의 톤(격식체와 따뜻함의 균형, "함께·연결·생태계" 키워드, 차갑지 않은 정중함)을 1순위로 둡니다.

---

## 1. 우리가 말하는 방식 — 다섯 가지 원칙

1. **사명을 먼저 말합니다. 사업은 그 다음에 옵니다.**
   "우리는 무엇을 한다"가 아니라 "누가 무엇을 위해 일할 수 있도록"부터 시작합니다.

2. **평어체로 정중하게, 차갑지 않게 씁니다.**
   "~합니다" 기본형. 단, 거리를 두는 격식어("당사", "귀사")는 쓰지 않습니다.

3. **쉬운 우리말을 먼저 고려합니다.**
   필요한 전문 용어(임팩트 조직, AI, 시스템화)는 그대로 두되, 그 외에는 일상어로 바꿉니다.

4. **"함께"의 자리를 만듭니다.**
   우리가 하는 일이 아니라 같이 하는 일로 씁니다. "대신 해드립니다"가 아니라 "같이 짭니다".

5. **문제 진단보다 가능성에 무게를 둡니다.**
   막힌 지점을 짚되, 비난·과장 없이 공감조로. 결론은 늘 "그래서 어떻게 풀어낼 수 있는가"로 이어집니다.

## 2. DO / DON'T 변환 예시

현재 홈페이지에서 가져온 실제 카피를 기준으로 정리한 변환표입니다.
"❌ → ✅"는 "고쳐야 한다"가 아니라 **"이런 결로 옮기는 것이 가이드 방향"**이라는 뜻입니다.

| 위치 | 현재 카피 | 가이드 방향 변환안 |
|---|---|---|
| 문제 정의 | ❌ "사람의 시간을 갉아먹는 반복 업무" | ✅ "정작 중요한 일에 시간이 머무를 수 있도록" |
| 문제 정의 | ❌ "사람과 함께 사라지는 노하우" | ✅ "한 사람의 머릿속에만 있던 일을, 모두의 자리로" |
| 문제 정의 | ❌ "도구는 많지만, 답은 보이지 않습니다" | ✅ "도구가 아니라 일하는 방식부터 함께 봅니다" |
| 정체성 | ❌ "운영 효율화 전문사" / "운영 파트너" | ✅ "임팩트 조직이 미션에만 몰입할 수 있도록 곁에서 일하는 팀" |
| 서비스 도입 | ❌ "단체 및 기업의 서비스 도입 문의, 운영 효율화 관련 상담" | ✅ "조직의 일하는 방식을 함께 다시 짜고 싶은 분들의 이야기를 듣습니다" |
| 매뉴얼화 | ❌ "개인의 '감'에 기대던 일을 누구나 따라할 수 있는 순서로 정리합니다" | ✅ (현행 유지 — 이미 가이드에 부합) |
| 워크샵 안내 | ❌ "효율보다 미션 기여도를 기준으로 일의 우선순위를 다시 세웁니다" | ✅ (현행 유지 — '효율' 대신 '미션'이 앞에 있어 결이 맞음) |

> 참고: 현재 카피 다수가 이미 가이드 결에 들어와 있습니다. 손볼 곳은 주로 **문제 정의 섹션의 첫인상**과 **정체성을 한 줄로 압축한 표현(헤더·메타 설명)**입니다.

## 3. 어휘 사전

### 선호어 (먼저 고려)

| 선호어 | 지양어 | 이유 |
|---|---|---|
| 함께 / 같이 | 우리가 / 저희가 단독 | 협력의 자리를 만든다 |
| 곁에서 | 외주로 / 대행으로 | 파트너십 톤 |
| 비웁니다 / 새로 짭니다 | 효율화합니다 / 최적화합니다 | 사명 중심 표현 |
| 미션 / 일의 본질 | KPI / 성과 지표 | 임팩트 조직의 언어 |
| 짚어보다 / 들여다보다 | 진단하다 / 분석하다 | 일상어 우선 |
| 자리 / 흐름 | 프로세스 / 워크플로 | 자연어 비유 |
| 머무르다 / 새다 | 소요되다 / 누수되다 | 부드러운 결 |
| 임팩트 조직 | 사회적기업·NGO·재단·공공기관 (모두 풀어쓸 때) | 한 단어로 묶을 때만 임팩트 조직 사용 |
| 도구 | 솔루션 / 플랫폼 | 위협감 줄이기 |
| 다시 쓰입니다 / 이어집니다 | 환원합니다 / 재투입합니다 | 흐름의 언어 |

### 그대로 살릴 표현 (이미 우리 자산)

- "낡은 일하는 방식을 함께 비우고, 새로 만듭니다"
- "끝까지 같이 짭니다"
- "조직에 맞지 않는 방식을 먼저 비우고, 그 자리에 꼭 필요한 도구만 남깁니다"
- "세상을 바꾸는 사람들이 미션에만 몰입할 수 있도록"
- "수익은 다시 사회로 / 다음 사람에게로 이어지도록"
- "채우기 위해서는 먼저 비워야 합니다"

### 한자어·외래어 점검 리스트 (한 번 더 들여다보고 결정)

- "도입" → 맥락에 따라 "들이다", "함께 만들다"로 풀 수 있음
- "운영" → 유지하되, "운영 파트너"는 "곁에서 일하는 팀"으로 대체 가능
- "DX 전환", "노코드", "온보딩" → 의미가 명확하므로 유지. 단, 첫 등장 시 한 줄 설명 권장

## 4. 메시지 순서 — 사명 → 사업 템플릿

새 글이나 새 섹션을 쓸 때 다음 4단계 흐름을 권장합니다.

### 1단계. 왜 (사명)
**누구를 위해, 왜 이 일을 하는가**를 한 문장으로 먼저 둡니다.
- 예: "세상을 바꾸는 일에만 몰입할 수 있도록"

### 2단계. 무엇이 막고 있나 (공감)
pain point 묘사가 아니라 **조직이 마주하는 자연스러운 상황**으로 씁니다.
비난·과장 없이, 듣는 사람이 "맞아, 우리 얘기네"로 끄덕이게.
- ❌ "갉아먹는", "새어 나가는" 같은 부정 동사의 반복은 한 곳에 한 번만
- ✅ "본업에 머물러야 할 시간이, 자꾸 다른 자리로 갑니다"

### 3단계. 어떻게 (방법)
**"비움 → 새로 짜기"** 흐름으로. AI·자동화·도구는 늘 뒤에, 방법론 다음에 옵니다.
- 1) 어떤 일이 진짜 중요한지 함께 정리
- 2) 일하는 방식을 다시 짜기
- 3) 거기에 맞는 도구만 남기기

### 4단계. 함께 (초대)
다음 단계로 부드럽게 안내합니다. "신청하세요"보다 **"이야기를 들려주세요"**의 결.
- 예: "어디서 시간이 새고 있는지 들려주세요"
- 예: "30분이면 가장 새는 자리가 보입니다"

## 5. 비유와 이미지

### 우리 비유의 코어 자산

| 비유 | 의미 | 사용 예 |
|---|---|---|
| **비움** | 낡은 방식을 내려놓는 일 (Unlearning의 직역) | "먼저 비우고, 그 자리에 꼭 필요한 도구만 남깁니다" |
| **자리** | 사람의 역할·시간·일의 위치 | "그 자리에", "막혀 있는 자리", "다음 사람의 자리" |
| **흐름** | 일이 막힘없이 이어지는 모습 | "시간이 새다", "흐름을 다시 내다" |
| **곁** | 옆에서 함께 일하는 관계 | "곁에서 함께 일합니다" |
| **다시 쓰다 / 이어지다** | 수익·노하우의 재순환 | "수익은 다시 사회로", "다음 사람에게로 이어지도록" |

### 비유 확장 방향 두 갈래 (선택)

- **A안. 일하는 풍경의 비유**: "비어 있는 책상", "다시 내는 물길", "막힌 길을 트다"
   - 추상적이지 않고 손에 잡힘. 기업 톤과 자연스럽게 어울림.
- **B안. 자연·생명력 비유**: 루트임팩트의 "징검돌"·"뿌리", 브라더스키퍼의 "초록빛 희망" 결
   - 임팩트 조직 톤에 더 가까움. 단, 우리 사업(시스템·AI)과의 거리는 의도적으로 좁혀야 함.

> **권장:** A안을 메인으로, B안은 어바웃·임팩트 섹션에서 보조로 사용. 두 갈래를 섞기보다 섹션별로 일관되게.

## 6. 카피 작성 체크리스트

새 카피를 쓰거나 기존 카피를 손볼 때 7문항을 점검합니다.

- [ ] 첫 문장이 "우리가 무엇을 한다"가 아니라 **"왜 / 누구를 위해"**로 시작하는가
- [ ] 한자어를 쉬운 우리말로 바꿀 수 있는 자리가 남아 있는가
- [ ] **"함께"의 자리**가 들어가 있는가 (남발하지 말되, 적어도 한 곳)
- [ ] 문제 진단이 비난조가 아닌 공감조인가 ("갉아먹는", "낭비되는"의 반복은 한 번까지)
- [ ] **도구·기술이 앞이 아니라 뒤**에 있는가
- [ ] 3~7어절의 **짧은 핵심 문장**이 한 줄 있는가 (스캔하는 독자를 위해)
- [ ] 추상 가치(미션·임팩트)와 **구체 사례**(회계 정산, 보고서, 정산서)가 한 묶음으로 붙어 있는가

## 8. 적용 순서 (다음 단계 가이드)

이 가이드가 승인되면 다음 순서로 카피 개정을 진행할 것을 권장합니다.

1. **메인 헤더·메타 설명**: 검색 결과·SNS 공유 시 첫 인상이라 우선
2. **히어로 섹션**: "낡은 일하는 방식을 함께 비우고, 새로 만듭니다"는 유지, 직전 보조 카피만 손봄
3. **문제 정의 3블록**: 가장 큰 변경 지점. pain → 가능성으로 결을 전환
4. **어바웃·핵심 가치 섹션**: 비유 강화, "다시 쓰이는 시간" 흐름의 언어 정리
5. **CTA·문의 페이지**: "신청"의 결을 "이야기를 들려주세요"로
